What's the dumbest code you've ever written ?
==
The C language exam asked me to write a program to calculate 1+2²+3² up to n², but the loop I wrote couldn't run. Finally, when I was about to hand in the paper, I wrote printf n(n+1)(2n+1)/6, and then I passed the exam.
